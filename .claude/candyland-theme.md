## Usage Notes

- **Primary actions**: Use pink (`#ffc0cb` light / `#ff99cc` dark)
- **Secondary actions**: Use sky blue (`#87ceeb`)
- **Highlights/Accent**: Use yellow (`#ffff00`)
- **Success states**: Use green (`#33cc33`)
- **Destructive actions**: Use red (`#ef4444`)
